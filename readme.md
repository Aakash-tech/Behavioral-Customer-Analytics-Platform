# 📊 Behavioral Customer Analytics Platform

A Streamlit-based interactive dashboard for Customer Segmentation using RFM (Recency, Frequency, Monetary) Analysis.

This project analyzes customer purchasing behavior and segments customers into meaningful business groups like High Value, Regular, Inactive, and At Risk customers.

---

## 🚀 Features

- 📈 RFM-Based Customer Segmentation
- 💵 Total Revenue & Customer KPIs
- 🏆 Top 5 Customers by Cluster
- 📊 Revenue Contribution by Cluster
- 🔎 Interactive Cluster Filtering
- 🎯 RFM Scatter Visualization (Recency vs Monetary)
- 🧠 Business-Friendly Cluster Labels

---

## 🧠 Cluster Definitions

| Cluster | Meaning |
|----------|----------|
| 🏆 High Value | High spending, frequent & recent buyers |
| 💰 Regular | Moderate spending customers |
| 😴 Inactive | Low frequency & low engagement |
| 🚨 At Risk | Previously active but not recent |
